Abandoned Mall

A Work-In-Progress Map built in S&box.

Learning how to build a level in the Source 2 Engine

Abandoned Mall is my most recent personal project, it is a map created for a Zombie Survival game mode in the Source 2 engine. It has been built from the ground utilizing the features of the S&Box developer preview.

The main objective of this project is to build a map in Source 2 utilizing level design techniques, lighting design, art direction and 3D modelling. This map is inspired by a Garry's Mod map of the same name made by WaterPyro in 2012.

All 3D modelling and design is done within Autodesk Maya and Adobe Substance 3D Painter and Photoshop. Textures have been provided by Poliigon.
