VaporWorld

This game was my part of a project for my Interface Design class. The assignment was to create a functional game that has an easy-to-understand UI.

The game also had to have an accurate scoring system; a calculation based on how many collectibles the player is able to find, and how many areas they visit within the game's time limit, minus the amount of steps taken.

The final product is a Vaporwave themed game where the player walks around, searching for the different collectables in the level, trying to increase their score before they run out of time (health). There is also a special powerup that opens a pop-up menu, allowing the player to gamble for extra score, as well as 4 distinct areas for the player to visit.

One of the constraints of the project was that the character had to use Unity's NavMesh system for movement. I wanted the player to use third person movement, so I had to write a script that allowed the player to move with their mouse and keyboard, but use NavMesh to map out the walkable areas for the player.

The game encapsulates the Vaporwave aesthetic by using many features of the genre. It uses retro shaders that make the game look like a stylish 80's movie being watched on a CRT TV. It also uses low-poly models, electronic music that crossfades as you enter different areas, and UI elements reminiscient of late 1990s web design.
