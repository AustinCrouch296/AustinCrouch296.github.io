VaporWorld

This game was created as a project for my Interface Design class at Bournemouth University. The assignment was to create a functional game that has an easy-to-understand UI.

The game tracks the player's progress with a scoring system; a calculation of how many collectibles the player found and how many areas they visited within the game's time limit, minus the amount of steps taken.

The final product is a Vaporwave-themed exploration game where the player walks the level, searching for the different collectibles in the 4 distinct areas they can visit, trying to increase their score before their health (time) runs out. There is also a special powerup that opens a menu, allowing the player to gamble for extra score.

The project was implemented through Unity, with the constraint that the character had to use the NavMesh system for movement. I wanted the game to have third-person movement, so I had to create a script that would detect if the location that the player was trying to move to was a valid NavMesh surface.

The game encapsulates the Vaporwave aesthetic by using many features of the genre. It uses retro shaders that make the game look like a stylish 80's movie being watched on a CRT TV. It also uses low-poly models, electronic music that crossfades as you enter different areas, and UI elements reminiscient of late 1990s web design.
